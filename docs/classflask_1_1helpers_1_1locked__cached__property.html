Detailed Description

A :func:`property` that is only evaluated once. Like
:class:`werkzeug.utils.cached_property` except access uses a lock
for thread safety.

.. versionchanged:: 2.0
    Inherits from Werkzeug's ``cached_property`` (and ``property``).
