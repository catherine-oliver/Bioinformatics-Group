Vue Test Utils

Currently in beta

To use Vue Test Utils beta:

// npm
npm install --save-dev @vue/test-utils
// yarn
yarn add --dev @vue/test-utils

Intro

Vue Test Utils is the official test library for Vue.js. It provides methods for unit testing Vue components.
