Detailed Description

Like the default :class:`UnicodeConverter`, but it also matches
slashes.  This is useful for wikis and similar applications::

    Rule('/<path:wikipage>')
    Rule('/<path:wikipage>/edit')

:param map: the :class:`Map`.
