Detailed Description

Like `Subdomain` but prefixes the URL rule with a given string::

    url_map = Map([
        Rule('/', endpoint='index'),
        Submount('/blog', [
            Rule('/', endpoint='blog/index'),
            Rule('/entry/<entry_slug>', endpoint='blog/show')
        ])
    ])

Now the rule ``'blog/show'`` matches ``/blog/entry/<entry_slug>``.
