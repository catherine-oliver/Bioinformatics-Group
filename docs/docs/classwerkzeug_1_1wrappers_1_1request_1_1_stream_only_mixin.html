Detailed Description

Mixin to create a ``Request`` that disables the ``data``,
``form``, and ``files`` properties. Only ``stream`` is available.

.. deprecated:: 2.0
    Will be removed in Werkzeug 2.1. Create the request with
    ``shallow=True`` instead.

.. versionadded:: 0.9
